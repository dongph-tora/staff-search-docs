🎯 概要

アプリケーション情報  
アプリ名: スタッフサーチ ヘッドハンティング管理  
パッケージ名: `com.stafffinder.headhunting_app`

プラットフォーム: Web  
ポート: 9090  
フレームワーク: Flutter 3.35.4  
Dart バージョン: 3.9.2

目的  
企業・店舗がスタッフにスカウトオファーを送信し、採用活動を行うための専用管理ツール。オファー管理、メッセージ機能、プラン管理などを提供する。

主要機能
- **スタッフ検索**: 条件別スタッフ検索・閲覧
- **オファー管理**: オファー作成・送信・ステータス管理
- **メッセージ機能**: スタッフとの直接メッセージ
- **プラン管理**: 有料プラン管理
- **お知らせ**: 運営からのお知らせ受信
- **通知**: オファー応答通知

class Company {
  String id;              // 企業ID

  String phoneNumber;     // 電話番号

  String? logoUrl;        // ロゴURL

  String benefits;        // 特典・福利厚生

🎯 ビジネスフロー

オファー送信フロー

```mermaid
flowchart TD
  A[企業: スタッフ検索] --> B[スタッフ詳細表示]
  B --> C[オファー送信ボタン]
  C --> D[オファーフォーム入力]
  D --> E{入力完了?}
  E -->|いいえ| D
  E -->|はい| F[オファー送信]
  F --> G[スタッフ通知]
  G --> H[ステータス: pending]
  H --> I{スタッフ返信}
  I -->|承諾| J[ステータス: accepted]
  I -->|拒否| K[ステータス: rejected]
  J --> L[企業通知: 承諾]
  K --> M[企業通知: 拒否]
  L --> N[メッセージ開始]
  M --> O[オファー終了]
```

プラン変更フロー

```mermaid
flowchart TD
  A[企業: プラン管理] --> B[現在のプラン確認]
  B --> C[プラン一覧表示]
  C --> D[プラン選択]
  D --> E[プラン詳細確認]
  E --> F{変更確認}
  F -->|キャンセル| C
  F -->|確認| G[決済画面]
  G --> H[決済処理]
  H --> I{決済成功?}
  I -->|失敗| J[エラー通知]
  I -->|成功| K[プラン切替]
  K --> L[完了通知]
```
