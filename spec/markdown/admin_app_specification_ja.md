🎯 概要

アプリケーション情報  
アプリ名: スタッフサーチ 運営管理パネル  
パッケージ名: `com.stafffinder.admin_app`

プラットフォーム: Web  
ポート: 8181  
フレームワーク: Flutter 3.35.4  
Dart バージョン: 3.9.2

目的  
スタッフサーチプラットフォームの運営管理機能を提供。ユーザー管理、コンテンツモデレーション、通知配信、レポート分析などを一元管理する。

主要機能
- **ユーザー管理**: ユーザー・スタッフ・店舗の管理
- **コンテンツモデレーション**: 投稿・ライブ配信の監視
- **ヘッドハンティング管理**: 企業・オファー管理
- **通知配信**: プッシュ通知・お知らせ管理
- **レポート分析**: 統計情報・収益分析
- **サポートチャット**: ユーザーサポート対応
- **SNS管理**: ソーシャルメディア連携

🎯 データモデル

`AdminUser`（管理者）

```dart
class AdminUser {
  String id;           // 管理者ID
  String email;        // メールアドレス
  String name;         // 表示名
  String role;         // ロール (admin/moderator)
  DateTime createdAt;  // 作成日時
  DateTime lastLoginAt;// 最終ログイン日時
}
```

`Notification`（通知）（NEW v1.2.0）

```dart
class Notification {
  String id;             // 通知ID
  String title;          // タイトル
  String body;           // 本文
  String? imageUrl;      // 画像URL
  String? linkUrl;       // リンクURL
  String target;         // 配信対象 (all/users/staff/stores)
  DateTime? scheduledAt; // 配信予定日時
  DateTime? sentAt;      // 配信日時
  String status;         // ステータス (scheduled/sent/failed)
  int recipientCount;    // 配信先数
  DateTime createdAt;    // 作成日時
}
```

`Announcement`（お知らせ）（NEW v1.2.0）

```dart
class Announcement {
  String id;             // お知らせID
  String title;          // タイトル
  String content;        // 本文
  String category;       // カテゴリ (important/maintenance/feature/event)
  String targetAudience; // 対象ユーザー (all/users/staff/stores)
  DateTime publishedAt;  // 公開日時
  String? imageUrl;      // 画像URL
  bool isPublished;      // 公開状態
  int viewCount;         // 閲覧数
  DateTime createdAt;    // 作成日時
  DateTime updatedAt;    // 更新日時
}
```
