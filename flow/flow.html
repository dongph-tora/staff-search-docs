STAFF-02 Auto-generate 6-digit Staff ID
Backend only — no dedicated screen Triggered by: StaffService.CreateProfile (STAFF-01) crypto/rand → zero-pad 6 digits → uniqueness check in staff_profiles → retry up to 10x on collision → store in staff_profiles.staff_number Displayed on: StaffProfileScreen as "Staff ID: XXXXXX"
STAFF-03 Job Category Selection
No dedicated screen — widgets only JobCategoryDropdown → embedded in StaffRegistrationScreen → embedded in StaffProfileEditScreen JobCategoryChips (horizontal filter row) → embedded in HomeScreen below AppBar → embedded in SearchScreen Data: GET /api/v1/staff/job-categories 21 categories · cached per session
